# scultura_3D_Object

Dieses Repository enthält ein einfaches Setup für dein 3D-Objekt-Projekt.

Struktur:
- models/        -> Hier kommt deine rock.glb (oder weitere Modelle)
- examples/      -> Beispielseite zum Laden des GLB mit three.js

Kurz: Lade deine rock.glb in den Ordner models/ (per Web-UI oder Git). Danach kannst du die Datei über jsDelivr oder raw.githubusercontent ausliefern.

Beispiel-raw-URL (nach Upload):
https://raw.githubusercontent.com/Muftie93/scultura_3d_object/main/models/rock.glb

Empfohlen (CDN):
https://cdn.jsdelivr.net/gh/Muftie93/scultura_3d_object@main/models/rock.glb

Anleitung zum Hochladen findest du in der Repo-UI oder per Git (siehe .gitignore und examples/README).
